# Policy/FAQ to Document Draft Evidence Pack

## When To Use This Template

Use this template when:
- You researched policy/regulation/FAQ pages and need to draft a document (memo, guide, summary, report)
- The next task is document drafting that depends on exact policy wording
- You need to preserve which points can be quoted verbatim, which need paraphrase, and which need re-check

## Input Context

- Research source: [policy page / FAQ / official guidance / regulation document]
- Target document type: [memo / guide / summary / report / explanation / Q&A]
- Audience: [internal team / external client / management / general]

## Evidence Structure

### 1. Verbatim-Ready Claims
Direct quotes or near-quotes that can be used as-is:

| Source | Section | Quote | Context Note |
|--------|---------|-------|--------------|
| [URL] | [heading] | "exact text" |适用场景说明 |

### 2. Paraphrase-Ready Points
Facts that need rephrasing but are source-backed:

| Point | Source | Original Wording | Paraphrase Direction |
|-------|--------|------------------|---------------------|
| 核心要点 | URL | 原文表述 | 改写方向 |

### 3. Re-Check Required
Points that need live verification before use:

| Point | Source | Why Uncertain | Trigger Condition |
|-------|--------|----------------|------------------|
| 待确认点 | URL | 不确定原因 | 何时需要重新核实 |

### 4. Missing Information
Gaps identified during research:

| Needed Info | Source Checked | Where to Find | Priority |
|-------------|----------------|---------------|----------|
| 缺失信息 | 已查来源 | 待查来源 | 高/中/低 |

## Downstream Routing

- For document drafting: → `document-intake-brief` → `document-task-routing`
- For internal memo: → specific format requirements
- For client summary: → audience-specific tone guide

## Usage Notes

1. Always verify "Re-Check Required" items before final document delivery
2. For regulatory documents, prioritize verbatim quotes over paraphrasing
3. When in doubt about policy interpretation, flag for human review

## Example

### Scenario: Research company's remote work policy for employee handbook

**Verbatim-Ready Claims:**
- "员工每周可申请最多2天远程办公" - 远程工作政策页第3条
- "需提前3个工作日提交申请" - 申请流程说明

**Paraphrase-Ready Points:**
- 申请流程需要直属上级审批 → 原文：需经部门负责人审批
- 远程办公设备要求 → 原文：公司提供标准配置

**Re-Check Required:**
- 节假日远程办公安排 → 政策页面未明确，需HR确认
- 跨城市远程申请 → 标注为"需单独审批"
